# Reflection & Feedback Loops in Agentic AI

## Definition
Reflection is the agent’s ability to **evaluate its actions**, learn from outcomes, and improve future performance.

## Key Points
- Feedback loops: Success/failure signals guide adjustments.
- Error handling: Retry or change strategy.
- Continuous learning makes agents more reliable.

## Example
An agent that retries a failed API call with a different parameter, then logs the error for future improvement.

## Interview Tip
Mention reflection as the **self‑improvement mechanism** of agents.
