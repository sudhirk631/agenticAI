# Reasoning & Planning in Agentic AI

## Definition
Reasoning and planning allow agents to **analyze context, set goals, and design multi‑step strategies**.

## Key Points
- Reasoning: Logical decision‑making based on inputs.
- Planning: Breaking down tasks into steps.
- Enables agents to handle complex workflows.

## Example
An agent that queries BigQuery, interprets results, and then **generates a report with recommendations**.

## Interview Tip
Emphasize that reasoning + planning make agents **goal‑oriented**, not just reactive.
