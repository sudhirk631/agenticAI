# Tool Use & API Integration in Agentic AI

## Definition
Agents extend their capabilities by **using external tools and APIs** to perform actions beyond content generation.

## Key Points
- Connect to cloud services (BigQuery, Cloud Storage, Vertex AI).
- Call external APIs for data retrieval or workflow automation.
- Tool use makes agents practical in real‑world systems.

## Example
An agent that **uploads files to Cloud Storage**, then **summarizes them using Vertex AI**.

## Interview Tip
Stress that tool use is what makes agentic AI **industry‑ready**.
